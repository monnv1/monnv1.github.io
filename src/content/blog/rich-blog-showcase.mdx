## 视频

可以直接写 HTML 的 `<video>`。这里用浏览器原生播放器做占位模板，你以后把 `src` 换成自己的视频文件即可。

```html
<video controls src="/videos/demo.mp4"></video>
```

也可以嵌入 B 站或 YouTube 的 iframe。

```html
<iframe src="https://www.youtube.com/embed/VIDEO_ID"></iframe>
```

## 动图

动图可以直接用 Markdown 图片语法：

```md
![演示动图](/images/demo.webp)
```

建议优先用 `.webp`，体积通常比 `.gif` 更友好。

## 代码高亮

Astro 自带代码高亮，普通 Markdown 代码块就能工作。

```ts
type PostKind = 'note' | 'essay' | 'demo';

const publish = (kind: PostKind) => {
  return `Publishing ${kind}`;
};
```

这套组合的重点是：日常写作保持简单，需要高级表达时再使用组件。
